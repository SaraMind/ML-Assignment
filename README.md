#  Dataset Description  

This project uses the **Student Mental Health Dataset** from Kaggle.

Dataset Source:  
https://www.kaggle.com/datasets/shariful07/student-mental-health  

---

#  Dataset Overview  

- Number of Rows: 100  
- Number of Columns: 11  

---

#  Features Description  

- Timestamp: Time when the response was recorded  
- Choose your gender: Gender of the student  
- Age: Age of the student  
- What is your course?: Academic program  
- Your current year of Study: Study year  
- What is your CGPA?: Academic performance range  
- Marital status: Single or married  
- Do you have Depression?: Depression status (Yes/No)  
- Do you have Anxiety?: Anxiety status (Yes/No)  
- Do you have Panic attack?: Panic attack status (Yes/No)  
- Did you seek any specialist for a treatment?: Whether professional help was sought

- Identify data quality issues
