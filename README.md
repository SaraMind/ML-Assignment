# EDA-Assignment
# Student Mental Health Dataset

## Dataset Source
This dataset was taken from Kaggle.
Link: https://www.kaggle.com/datasets/shariful07/student-mental-health

---

## Number of Rows and Columns
- Number of Rows: 100
- Number of Columns: 11

---

## Features Description

- Timestamp: Time when the response was recorded  
- Choose your gender: Gender of the student  
- Age: Age of the student  
- What is your course?: Academic course or program  
- Your current year of Study: Current study year  
- What is your CGPA?: Student academic performance range  
- Marital status: Whether the student is single or married  
- Do you have Depression?: Depression status (Yes/No)  
- Do you have Anxiety?: Anxiety status (Yes/No)  
- Do you have Panic attack?: Panic attack status (Yes/No)  
- Did you seek any specialist for a treatment?: Whether student sought professional help  

---

## Purpose of the Dataset

This dataset is used to analyze the relationship between students’ academic performance and mental health conditions such as depression, anxiety, and panic attacks. The goal is to explore patterns and insights that may help understand student well-being.

---

## Summary

This project includes data cleaning, visualization, and exploratory data analysis (EDA) to extract meaningful insights from the dataset.
